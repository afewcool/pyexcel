# 엑셀 보다 파이썬

1. 압축 파일(zip 파일) 다운로드
    - 아래 다운로드 위치에서 압축 파일을 다운로드 합니다.
    - 깃허브 저장소: https://github.com/wikibook/pyexcel
    - ZIP 형식으로 다운로드: https://github.com/wikibook/pyexcel/archive/master.zip

2. 압축 풀기
    - 다운로드 받은 압축 파일을 임의의 폴더에 풉니다.
    - 압축을 풀면 `data`, `figures`, `notebooks` 폴더가 있습니다.

3. 폴더 복사
    - 윈도우 탐색기로 작업 폴더인 `C:\myPyExcel`를 생성합니다.
    - 압축을 푼 폴더에서 `data`, `figures`, `notebooks` 폴더를 모두 작업 폴더(여기서는 `C:\myPyExcel`에 복사합니다.)

    - 데이터 파일 복사: data 폴더의 내용 복사
      - data 폴더에는 이 책의 코드에서 사용하는 데이터 파일이 있습니다.
      - 다운로드 받은 data 폴더 내의 내용을 자신이 이용하는 작업 폴더 내의 data 폴더(여기서는 C:\myPyExcel\data)에 복사합니다.
    - 그림 파일 복사: figures 폴더의 내용 복사
      - figures 폴더에는 이 책의 코드에서 사용하는 그림 파일이 있습니다.
      - 다운로드 받은 figures 폴더 내의 내용을 자신이 이용하는 작업 폴더 내의 figures 폴더(여기서는 C:\myPyExcel\figures)에 복사합니다.
    - 노트북 복사: notebooks 폴더의 내용 복사
      - notebooks 폴더에는 이 책에서 사용한 코드가 담긴 노트북 파일(\*.ipynb)이 있습니다.
      - notebooks 폴더에 있는 노트북 파일을 자신의 작업 폴더(여기서는 C:\myPyExcel)에 복사합니다.

4. 노트북의 코드 셀 실행
    - 주피터 노트북이 실행되지 않은 상태라면 아나콘다 메뉴에서 [Anaconda Prompt] 클릭해서 아나콘다 프롬프트를 실행합니다.
    - 아나콘다 프롬프트에 명령어(만약 작업 폴더가 C:\myPyExcel라면 `'cd C:\myPyExcel'`)를 입력해서 작업 폴더로 이동합니다.
    - 아나콘다 프롬프트에 `jupyter notebook`을 입력합니다.
    - 주피터 노트북이 실행되면 브라우저의 홈 화면(Home page)에 보이는 노트북 파일에서 필요한 노트북을 클릭합니다.
    - 예제 코드가 있는 코드 셀을 실행([셀 실행] 아이콘을 클릭하거나 키보드의 `Shift + Enter` 키 입력)합니다.
